This application is designed to fit mid-companies. Also, the system contains of more than many sections

- General settings
- The products
- Store settings
- Categories
- Roles
- Home User
- Checkout
- Pay User
- Complete Pay
- Cart User
- Payment Succed
- Product User




This Application programmed by PHP, Laravel , JS, JQuery, HTML,CSS,AJAX
